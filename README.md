# Factors of success in abstract theoretical course — ML Analysis

A machine learning project analyzing student engagement patterns in an online university course and predicting final grades based on activity data. The dataset spans two years of the same course (Year A and Year B), enabling cross-year comparison of behavioral patterns and model performance.

## Dataset

The data comes from a university course delivered over two semesters. Students could attend lectures live online, watch recordings, complete 12 homework assignments, and take a midterm and final exam — all tracked in the activity log. The course materials were available in both English and Estonian.

| File | Description |
|------|-------------|
| `activity_log_A.csv` | Student activity events for Year A (timestamps, student IDs, activity names) |
| `activity_log_B.csv` | Student activity events for Year B |
| `grades_A.csv` | Homework scores, midterm, final exam, total points and grade for Year A |
| `grades_B.csv` | Same for Year B (also includes bonus points, which only existed in Year B) |
| `df_A.csv` | Preprocessed feature matrix for Year A (output of `preprocessing.ipynb`) |
| `df_B.csv` | Preprocessed feature matrix for Year B |

Grading scale: A ≥ 90, B ≥ 80, C ≥ 70, D ≥ 60, E ≥ 50, F < 50. Students who did not participate received a grade of `-`.

## Preprocessing

`preprocessing.ipynb` transforms raw activity logs into a clean feature matrix per student:

- **Homework scores**: English and Estonian versions merged by taking the maximum score
- **Activity aggregation**: Per-student event counts pivoted into feature columns
- **Feature consolidation**:
  - `Total Joined Lectures` — sum of all live lecture join events
  - `Started Homework` — total homeworks opened (across all 12)
  - `Submitted Homework` — total homeworks submitted
  - `Practiced submitting the homework` — use of the technical dry-run submission tool
  - `Viewed lecture slides` — total slide views across all weeks
  - `Viewed study materials` — all written material views (English + Estonian combined)
  - `Viewed this year lecture recording` — current-year recording views
  - `Viewed past year lecture recording` — archive recording views
  - `Viewed sample solutions of homeworks or exams`
  - `Viewed feedback on homeworks or exams`
  - `Viewed exam tasks after exam`
  - Additional: announcements forum reads, course info page views, dictionary views

The final feature matrices (`df_A.csv`, `df_B.csv`) merge activity features with grade data and are used directly in `MLProject.ipynb`.

## Analysis

`MLProject.ipynb` covers the following:

### Correlation Analysis
Heatmaps and regression plots of each activity feature vs. total score.

**Top correlated features with total score:**

| Feature | Year A | Year B |
|---------|--------|--------|
| Submitted Homework | 0.70 | 0.66 |
| Started Homework | 0.67 | 0.68 |
| Bonus | — | 0.52 |
| Viewed study materials | 0.46 | 0.37 |
| Viewed feedback | 0.34 | 0.42 |
| Total Joined Lectures | 0.32 | — |

### ANOVA
One-way ANOVA on each feature vs. total score. Key finding: **Submitted Homework** was the only statistically significant predictor (p ≈ 0.0018).

### Multi-Linear Regression (OLS)
Predicting total score from activity features. R² = 0.56 for Year A. **Submitted Homework** showed the strongest significant coefficient (≈3.78 per additional submission, p = 0.002).

### PCA
Principal Component Analysis to identify which features explain the most variance. Top contributors: **Viewed study materials, Started Homework, Submitted Homework, Viewed sample solutions, Viewed feedback**.

### Random Forest
`RandomForestRegressor` trained on binary grade (A/B vs. rest). Feature importance charts and prediction error distribution plotted for both datasets.

### Logistic Regression
Binary classification (A/B grade vs. lower). Best feature combinations found via exhaustive search. Standout feature in Year A: **Viewed sample solutions** (p = 0.031). In Year B: **Bonus** (p = 0.0057, coefficient = 1.63).

### XGBoost
`XGBClassifier` trained for grade prediction.

**Top features by F-score:**

| Rank | Year A | Year B |
|------|--------|--------|
| 1 | Viewed sample solutions (328) | Viewed study materials (299) |
| 2 | Viewed feedback (221) | Read announcements forum (268) |
| 3 | Submitted Homework (221) | Viewed feedback (262) |

### Ensemble Models
Two ensemble strategies applied to both datasets:
- **Stacking Classifier**: RandomForest + XGBoost + Logistic Regression with XGBoost meta-model. Feature importance visualized per base model and for the meta-model. Evaluated with 5-fold cross-validation.
- **Voting Classifier**: Soft voting across the same three base models. Combined feature importance computed as average across models.

## Key Findings

- **Submitting homeworks** is the single strongest behavioral predictor of final grade across both years and methods.
- **Starting homeworks** (even without submitting) also correlates strongly — suggesting engagement intent matters.
- **Passive consumption** (watching recordings, browsing past exams) shows weak or no significant relationship with grade.
- **Bonus tasks** (Year B only) showed a surprisingly strong effect in logistic regression, suggesting motivated students self-select into bonus work.
- Cross-year consistency: top features are stable across Year A and Year B, supporting generalizability.
